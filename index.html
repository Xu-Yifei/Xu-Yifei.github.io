projects

Here are some cool projects:

Firemap   GitHub

Firemap is an interactive map that utilizes Google Maps API and OpenWeatherApp API to display weather information in real-time. It also tells you how high the risk of fire is for any place on Earth. This was a collaborative project completed at the NASA 2018 Space Apps Hackathon.

Sequence Game   GitHub

Sequence Game is a very simple memory-testing game powered by p5. The player is tasked with memorizing a random chain of four different possible directions: up, right, down, left. Please note that this game has alarmingly crude graphics.
